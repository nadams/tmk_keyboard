NODE keyboard firmware
======================
Custom keyboard made by NODE-3

Build
-----
Move to this directory then just run `make` like:

    $ make -f Makefile.lufa

### 1. ANSI
This is the default keymap.

See [keymap_ansi.h](keymap_ansi.h) for detail.
